前言
#

今天來改造前陣子入手的 GameBoy Advance (GBA)。

GBA 是我蠻喜歡的一台主機，算是我小時候回憶的一部份。不過，近年來二手 GBA 的二手行情水漲船高，在日本品相不錯的機器也要賣到 10000 日幣左右，這個價格都可以抵一台二手 switch lite 了。

當前 GBA 改屏比較主流有兩種方案，一種是改 IPS 螢幕，另一種是用 NDSi 螢幕改成 GBA 螢幕，其後者好處為真正的點對點顯示。

廣納網友們的意見，我最終還是決定改 IPS 螢幕，原因是 NDSi 久了會有螢幕發黃的問題。 如果沒有發黃問題的話，我會選 NDSi 螢幕改屏，因為他是真正的點對點。

我是從淘寶上購入「數碼舞館」的全貼合 IPS 改裝套件，包含外殼。 這個螢幕除了可以觸控調亮度外，焊接對應點位後還可以做到長按 SELECT 呼出選單，更改各種濾鏡，啟動關閉模擬點對點效果等等。

安裝螢幕
#

所有改裝的零件如圖所示，看起來不會太複雜。螢幕排線的部分 32pin 與 40pin 都有提供（GBA分兩版本），焊接用的線也有附贈。

卸下後蓋的所有螺絲後就可以直接取下後蓋，也沒有什麼卡榫結構。然後再卸下主板上的螺絲，以及上方的螢幕排線，就可以將整塊主板取出。

不知為何我覺得 GBA 主板蠻漂亮的？ 可能是因為白色吧！哈哈哈。而且相較新機器構造簡單，一塊主板所有元件都在上面，也不會有很多排線要拆接。

接著我嘗試將排線與螢幕都接上，感覺很不錯！我有先稍微裝上外殼測試一下，確認沒有問題。

焊接。按鍵呼出功能
#

把外殼、按鍵與主板等裝回去，接上排線就行了…嗎？ 才怪啦～

等我全部裝完後，測試了一下螢幕觸控調光可用，很棒！但此時我才想起來，忘了焊接呀！這樣就沒法用 SELECT 按鍵呼出設定選單了。

所以我只好又把所有都拆開，然後補焊上了點位。由於菜單的邏輯是用 SELECT 控制選單開啟關閉，L 與 R 鍵控制選項，因此總共有三條線要焊。

排線上有 L R S 的標示，要焊接到的位置如圖所示。

焊好後，再次把外殼等裝回去。由於是全貼合螢幕，會讓螢幕與主板間多一個很大的空間，為此需要裝上螢幕支架墊高螢幕。

接著是將觸控的感應區域黏上。

開機測試一下。長按 SELECT 成功呼出設定菜單。觸控的部分，長按可以呼出設定選單與切換項目，短按可以讓數字增加。

淘寶店裡面是有一張圖說明選單內容，但我似乎是拿到更新的版本，多了一點點功能，下面我用粗體標出。

經過我實際測試，選單部分詳細設定如下：

  • BRT: 螢幕亮度 [15 檔]
  • CLR: 色彩模式 [5 檔] > 標準、高曝光、低飽和（看起來灰灰的）、黑白、復古綠
  • DSP: 顯示模式 [3 檔] > 標準、其他、點對點
  • FRM: 補充幀 [ON/OFF]
  • CG: 色彩強化 [3檔] >（我不知道怎麼描述，感覺就是不同顏色的輕微濾鏡）

完成
#

這樣整體改裝就算完成了。我們還插卡帶試試。

太棒了！顯示效果真的很不錯。比之前沒有背光時的狀態好太多。

目前唯一的問題就是左右兩片（手托？）發黃，我之後再去買零件來替換。另外就是按鍵跟外殼都是淺色的，之後一併把按鍵換掉。

除了改螢幕外，內建喇叭也可以替換掉，但我對原裝喇叭還算滿意，所以暫時沒有替換需求。電池的部分也可以購買 type c 的充電套件，這樣就可以改充電式的了！我目前是用索尼的充電電池。

本次改裝狀況就是以上，感謝瀏覽！
